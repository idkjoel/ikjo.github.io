GOING BAAAACK TO FLAT.

PROJECT DETAILS

PLATFORM: iOS / macOS

STATUS: In Development

VERSION: 0.4 (beta)

CATEGORY: Design / Iconography

ONLINE: iOS / macOS

With eMinor, the goal was to be able to produce a high quality, consistent icon set that followed the trend of rounded edges and flat colours. Although it is still in early development, eMinor provides a number of customization options to make the iOS experience a more friendly one.

eMinor includes a number of customization options, including status bar themes, lockscreen widgets, icons (with variations of the icons), and more. The goal is to make things appear more cohesive and more modern in the UI while keeping the cornerstone characteristics of iOS. This is achieved by keeping the general arrangement of the icons, keeping a consistent colour pallet, and keeping the UI of the actual applications familiar for users. Making tweaks to easily identifiable shapes allows for the user to understand the premise of a themed element, even if it is completely different from what the original looks like.
